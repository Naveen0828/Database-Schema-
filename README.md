# Library Management System (SQL)

Basic SQL project demonstrating CRUD, JOINs, stored procedures for library operations.

## Tech Stack
- MySQL/PostgreSQL
- VS Code / Oracle SQL Developer

## Setup
1. Run `setup.sql`
2. Execute `queries.sql` for operations
3. Test `advanced.sql` procedures

## Features
- Book/member/employee management
- Issue/return tracking
- Overdue analysis with fines
- Category reports
